This is the game logic for a Unity blackjack game that teaches the player to count cards using the Hi/Lo KO system.

The full project is too large to be uploaded to Git.

Avaliable now on the Google Play Store: https://play.google.com/store/apps/details?id=com.binarymakers.count21
